# 第一版
## 思路：
通过0-5进行异或运算，发现0-5的异或结果与删掉一个数字之后的异或结果，再做异或就是删掉的那个数字的值。（删掉的数字A对结果的影响：A二进制为0的列地方不变，为1的列地方为相反数）
```
/**
 * @param {number[]} nums
 * @return {number}
 */
var missingNumber = function(nums) {
    let ans = 0
    let re = 0
    for(let i = 1;i<nums.length+1;i++){
        re ^= i
    }
    for(const num of nums){
        ans ^=num
    }
    ans ^=re
    return ans
};
```
